代码运行模式

为了方便快速开发迭代和测试，xasset 提供了以下几种代码运行模式（ScriptPlayMode）：

  • 仿真模式
  • 预加载模式
  • 增量模式

以上几种模式都可以在设置中切换，如下图：

除了仿真模式外，都需要打包后才能运行。对于增量模式，本地可以先将打包后的资源部署到 hfs 或其他 web 服务器中测试，另外可以在版本管理中拷贝需要的版本的资源到安装包资源目录测试。了解设置中其他属性的作用，请参考设置的文档。

仿真模式​

编辑器可以跳过打包快速运行，减少漫长的打包等待时间。

预加载模式​

编辑器或真机都可以 0 编码 1 键关闭更新，直接加载本地既有的打包后的资源。

增量模式​

编辑器可以还原真机加载热更环境，线上问题可以在本地快速还原现场和处理。
